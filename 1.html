HTML

HTML이란 무엇인가? is Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

HTML identifies which parts of text are body paragraphs, headings, hyperlinks, bulleted/numbered lists, block quotes, italicized, bolded, etc., and the CSS determines how those parts look visually on the frontend.
